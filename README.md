# The Unseen Axis

ブラウザの中で、4次元形状の「3次元断面」と「3次元への投影」を観察するインタラクティブ作品です。

## 遊ぶ

**[The Unseen Axisを開く](https://83bite.github.io/4DSDF_v2/)**

インストールは不要です。PCまたはスマートフォンのモダンブラウザで遊べます。

## 遊び方

- 画面下部から観察したい形状を選びます。
- `W`スライダーを動かすと、4次元方向の位置が変わり、現れる3次元断面が変化します。
- 中央の形状をドラッグすると、現在の断面を好きな角度から眺められます。
- `AUTO`を有効にすると、形状ごとに調整された動きを自動で観察できます。
- `調整する`から、断面表示と投影表示、4次元回転、解像度などを変更できます。

## 収録形状

- Portal Loom
- Twin Worlds
- Tesseract
- Hypersphere
- Duocylinder
- Soft Fusion
- Hyper Cross
- Hyperplane Scan

各形状には、使用している定義式と短い説明を表示しています。

## この作品について

形状の計算と描画はブラウザ内で行われます。入力した数式や操作内容が外部へ送信されることはありません。
